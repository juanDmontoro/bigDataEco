Big data in Economics

 

Part 1 Presentation: Statistical learning

Juan D. Montoro-Pons

Academic year 2024/25

What’s in the guide

 

“Central to this course is the integration of machine learning techniques, which are essential for analyzing and extracting insights from both structured and unstructured Big Data. This course will demonstrate to students the great possibilities that the use of such data, powered by machine learning, provides for the study of topics in economics.”

Goals

 

  • Provide a practical introduction to machine learning techniques

  • Get acquainted with open-source software and specific libraries for machine learning tasks

  • Develop reproducible research practices using notebooks, ensuring that analyses are transparent, easily shareable, and can be replicated by others

Content

 

  1. Introduction to statistical (machine) learning
  2. Supervised learning (i): regression
  3. Supervised learning (ii): classification
  4. Additional topics: unsupervised learning, unstructured data, ML and causal inference

Teaching methodology

 

Classes will use an applied (hands-on) approach to statistical learning:

  1. Mix of theory and actual implemention of techniques

  2. Reference language: Python

  3. Emphasis on the reproducibility (use of jupyter notebooks) and interpretability of results

  4. Classroom discussion and participation will be encouraged

  • Team project presentation

Details
Overview A team project to be presented at the end of the course (final session of the semester).

Some aspects to consider:
    - Data preprocessing.
    - Model selection and justification.
    - Evaluation metrics and their interpretation.
    - Visualizations.
    - Presentation skills and clarity.
Topics Predicting the popularity of tracks on Spotify using artist and track features.
Predicting claims (risk or euro amount) in an auto insurance portfolio based on car/insured features.
Predicting the probability of failures from firm-level financial accounts.
