1.2 Tuples

Tuples in Python are immutable (meaning their elements cannot be modified once defined) ordered collections of elements. They are similar to lists but have some key differences due to their immutability, which makes them particularly useful in specific scenarios. Tuples are created similarly to lists, but with parentheses instead of brackets.

Tuples are ideal for storing data that should remain constant throughout the program. For example, storing coordinates or days of the week (things that should not change).

Tuples allow for unpacking multiple values into variables easily, making simultaneaous assignments in just one line of code. The example below assigns each element of a tuple by unpacking it to three newly defined variables x, y and z:

Key Uses of Tuples 1. Data That Should Not Change 2. Function Returns: tuples are often used to return multiple values from a function. 3. Keys in Dictionaries (ssee below) 4. Unpacking (tuples allow for unpacking multiple values into variables easily).

The main advantages of using tuples over lists is that tuples are generally faster than lists due to their immutability and simpler internal structure, consume less memory, making them suitable for large collections of constant data and allow for safe iterations (tuples prevent accidental overwriting during iteration or function calls).

1.3 Dictionaries

A dictionary in Python is a mutable, unordered collection that maps keys to values. It is one of the most widely used data structures due to its efficiency and versatility.

Dictionaries allow for key-value pair storage. This makes them appropriate for fast lookups, that is, efficiently retrieve data by keys, making dictionaries ideal for implementing look-up tables or caches.

Dictionaries can be modified in place, making them suitable for dynamic datasets. Furtermore built-in methods such as .get(), .keys(), .values(), and .items() provide convenient ways to access, manipulate, or iterate through the dictionary.

Comparison of basic data structures

Feature Dictionary List Tuple

2. NumPy’s arrays

NumPy (short for Numerical Python) is a fundamental library for scientific computing in Python. It provides support for creating and manipulating large, multi-dimensional arrays and matrices, along with a comprehensive collection of mathematical functions to operate on these data structures.

NumPy offers a core data structure ndarray (N-dimensional array) implemented in C. This implementation is key to NumPy’s performance and efficiency.

2.1.2 Creating a vector with constant spacing

The np.arange() function generates sequences of numbers with specified starting, stopping, and stepping values, making it convenient for evenly spaced values.

This functionality is particularly useful in numerical simulations, where you may need to define ranges of values to iterate over.

2.1.3 Generating Sequences with linspace and logspace

NumPy provides additional functions to generate sequences of numbers, which are particularly useful for scientific computations. np.linspace()

The np.linspace() function generates evenly spaced numbers over a specified range. Unlike np.arange(), it specifies the number of points rather than the step size.

This is particularly handy for plotting functions or when you need a predefined number of points between two bounds.

The np.logspace() function generates numbers evenly spaced on a logarithmic scale. You define the start and stop powers of 10, and NumPy generates values between 10start10start and 10stop10stop.

This function is widely used in scientific contexts, for example, when analyzing data that spans several orders of magnitude.

2.1.3 Creating a matrix

NumPy arrays extend naturally to two or more dimensions, which makes them ideal for creating and manipulating matrices.

Beyond manually specifying matrices, NumPy offers utility functions for creating arrays with specific patterns:

- Zeros Matrix: np.zeros([i, i]) creates a ixj matrix filled with zeros.
- Ones Matrix: np.ones([i, j]) creates a ixj matrix filled with ones.
- Identity Matrix: np.eye(i) creates a ixi identity matrix.

These are highly efficient for initializing arrays in numerical models.

3.6 Grouping across values of a variable: use of groupby

groupby is a powerful method in pandas for performing split-apply-combine operations. It groups data based on specified keys and allows you to apply aggregation functions to each group.

Note that after applying groupby in pandas, you can use a wide range of summary (aggregation) functions to compute statistics for each group. Here are some commonly used summary statistics.

Basic Aggregation Functions: these functions compute basic statistics for each group.

Function Description
.sum() Sum of values in each group
.mean() Mean of values in each group
.median() Median of values in each group
.std() Standard deviation of each group
.var() Variance of each group
.min() Minimum value in each group
.max() Maximum value in each group
.count() Number of non-null values in each group
.size() Total number of entries in each group

It is always possible to use .agg() to apply multiple functions or custom functions to a grouped object.

You can also use custom lambda (anonymous) functions:

It is also possible to apply Transformation Functions, which return values aligned with the original DataFrame instead of reducing them.

Function Description
.transform() Transform values by applying a function to each group
.cumsum() Cumulative sum within each group
.cummax() Cumulative maximum within each group
.rank() Rank elements within each group

Filtering and Boolean Aggregation functions help filter groups or compute Boolean aggregates.

Function Description
.filter() Filters groups based on a condition
.any() Checks if any value in the group satisfies a condition
.all() Checks if all values in the group satisfy a condition

Finally, for more advanced analysis, you can compute statistical properties:

Function Description
.quantile() Computes quantile values (e.g., .quantile(0.75))
.skew() Measures skewness of distribution within each group
.mad() Mean absolute deviation
.corr() Pairwise correlation of columns within groups
.cov() Pairwise covariance of columns within groups

It is also possible to combine the use of .agg() with a dictionary to compute multiple statistics for different columns.
